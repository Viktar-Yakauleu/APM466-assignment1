# APM466-assignment1
APM466 Assignment 1: The Value of Time

This repository contains the following files:

- APM466_Assignment_1_Code_Viktar_Yakauleu.ipynb.
This file contains all of the code used for this assignment. Please feel free to change the path of the file in the "Import File with Data" section.

- APM466 A1 10 Bonds - Viktar Yakauleu.csv.
This file contains all of the source data for the 10 selected bonds. It is a simplified and re-formatted version of the data for all bonds.

- APM466 A1 All Bonds.csv.
This file is the original document containing the data for all bonds. The 10 bonds are selected from this data. This file itself is not used as the source data in the code.
